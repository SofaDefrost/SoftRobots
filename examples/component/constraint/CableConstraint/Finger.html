Cable controlled by keyboard

In this scene we demonstrate how to use the CableConstraint component to model an inelastic 1d cable. This cable is then connected to a mechanical object made of an elastic material. A script is used to pull/release the cable which in return deforms the elastic object.

How to use the demo

After clicking on "Animate" you can click in the 3d view then pulling/releasing the cable by pressing CTRL+/CTLR-.

In this scene we are also using the PythonScriptController component.
